Выполняет поиск заданного слова в указанном файле.

Функция ищет заданное слово в содержимом файла и выводит результат поиска. В зависимости от параметра option, используется одна из двух реализаций:

  • Опция 1: Использование указателей на char (C-строки).
  • Опция 2: Использование строк C++ (std::string).
Parameters
wordСлово, которое необходимо найти в файле.
filenameИмя файла, в котором будет происходить поиск.
optionОпция выбора реализации поиска (1 - для C-строк, 2 - для std::string).
